# Task Manager System

Система управления задачами с интеграцией Telegram-уведомлений.

## 📋 Функциональность

### Основные возможности
- Создание и управление задачами
- Ролевая система (Администратор/Пользователь) 
- JWT аутентификация
- Уведомления в Telegram
- Напоминания о дедлайнах

### Роли пользователей
- **Администратор**: Полный доступ к системе, создание задач, назначение исполнителей
- **Пользователь**: Просмотр своих задач, изменение статуса выполнения

### Telegram уведомления
- Уведомление о назначении новой задачи
- Напоминание о приближающемся дедлайне
- Уведомление администратору об изменении статуса задачи
- Персональные уведомления в личные сообщения

## 🛠 Технологический стек

### Бэкенд
- **Node.js** + **Express.js** - серверная платформа
- **MySQL** + **Sequelize ORM** - база данных
- **JWT** - аутентификация
- **bcryptjs** - хеширование паролей
- **node-cron** - планировщик уведомлений
- **axios** - HTTP клиент для Telegram API

### Фронтенд
- **React** - пользовательский интерфейс
- **Vite** - сборка проекта

## 🚀 Установка и запуск

### Предварительные требования
- Node.js 14+
- MySQL 5.7+

### 1. Настройка базы данных

```sql
CREATE DATABASE task_manager;
CREATE USER 'task_user'@'localhost' IDENTIFIED BY 'task_password';
GRANT ALL PRIVILEGES ON task_manager.* TO 'task_user'@'localhost';
FLUSH PRIVILEGES;
```
### 2. Настройка бэкенда

```bash
cd backend
npm install
```


#### Инициализация базы данных:

```bash
npm run init-db
```

#### Запуск бэкенда:

```bash
npm start
```

#### Запуск бэкенда:

```bash
cd ../frontend
npm install
npm run dev
```

## 📡 API Endpoints

### Аутентификация

- ```POST /api/auth/register``` - Регистрация
- ```POST /api/auth/login``` - Вход в систему
- ```POST /api/telegram/bind``` - Привязка Telegram аккаунта

### Задачи

- ```GET /api/tasks``` - Получить задачи
- ```POST /api/tasks``` - Создать задачу (admin only)
- ```PATCH /api/tasks/:id``` - Обновить задачу
- ```DELETE /api/tasks/:id``` - Удалить задачу (admin only)

### Пользователи

- ```GET /api/users``` - Список пользователей (admin only)

## 🔧 Настройка Telegram бота

1. Создайте бота через @BotFather в Telegram
2. Получите токен и добавьте в .env файл
3. Начните диалог с ботом для получения chatId
4. Используйте endpoint /api/telegram/bind для привязки аккаунта
